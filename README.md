## Featured project

### Blackline
A privacy-first document redaction and release app.

Blackline supports:
- **Redact mode** for legal, public release, disclosure, and compliance use cases
- **Substitute mode** for enterprise AI reuse, where sensitive terms are replaced with safe semantic substitutes to preserve usefulness

Current MVP scope includes:
- `.txt` and `.md` support
- AI-assisted redaction proposals
- manual redaction
- manifest and export flow
- early Midnight preview direction
